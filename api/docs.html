Doetoez AI Assistant

Advanced AI assistant for coding, content creation, and problem solving with natural language processing

GET /api/ai/doetoez Ready
Intelligent AI assistant powered by advanced language models. Capable of code generation, content creation, problem solving, and natural conversations.
Query Parameters
Parameter Type Required Description
q string Required Your question or message for the AI assistant
apikey string Required Your API authentication key
Example Request
fetch('/api/ai/doetoez?q=How to create a music player with JavaScript&apikey=ndiidepzXmusic')
  .then(response => response.json())
  .then(data => {
    console.log('AI Response:', data.response);

Get random high-quality Blue Archive character images. Perfect for wallpapers, avatars, or content creation.
Query Parameters
Parameter Type Required Description
category string Optional Specific character category (e.g., "aris", "hina")
quality string Optional Image quality: "hd", "fullhd", "4k"
apikey string Required Your API authentication key
